=====ArtStation=====
Реализовать загрузку картинок или ссылок на картинки для вывода посредством
Ajax + JSON
Постановка задачи:
реализовать страницу-галерею с получением изображений под средством AJAX запросов с использованием Promise.
Данные об изображениях должны храниться в формате JSON на web-сервере.
Порядок выполнения:
Сформировать JSON-файл с данными об изображениях (все из вариантов):
“клиентский путь” до файла на сервере
url на внешнем ресурсе
base64
Разработка структуры HTML-документа
Реализовать AJAX запросы с использованием Promise
Реализовать отображение изображений в структуре документа
